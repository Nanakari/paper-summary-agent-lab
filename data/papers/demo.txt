Large language model agents have shown strong potential in solving complex tasks by combining reasoning, tool use, and memory. However, existing agent systems often suffer from unreliable tool selection, hallucinated intermediate reasoning, and lack of systematic evaluation. This paper proposes a modular agent framework that separates task planning, tool execution, memory management, and verification. Experiments on web navigation and knowledge-intensive question answering benchmarks show that the proposed framework improves task success rate and reduces tool-use errors compared with standard prompting baselines.
